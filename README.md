![LOGIN](https://user-images.githubusercontent.com/96362469/198583274-f32cb166-0280-406a-bf73-72c4e7a2b39f.png)

## :hammer: Functionalities

- `1`: User registration and authentication
- `2`: User profile with post list
- `3`: Create post
- `4`: List posts (Feed)
- `5`: User Edition
